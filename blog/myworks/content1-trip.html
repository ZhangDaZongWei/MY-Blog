    css样式

    我设置的总盒子的最大宽度为540px。第一这里要注意的是每个拥有col类的子元素都要设置固定的宽和高，不然会根据里面的内容张开不相等的盒子，具体原因也不清楚,另外还要设置各占相同的等分;第二注意设置拥有col类的子元素自动换行，且拥有r类的孙元素占父元素的整宽。下面的css代码只写主要的部分。

         .row {
        display:-webkit-flex;
        }
     .row .col {
        display:-webkit-flex;
        flex-wrap:wrap;
        flex-grow:1;
        width:180px;
        height:90px;
             }
     .row .col .r {
        flex-basis:100%;
                 }

    轮播部分

    自适应轮播部分注意：第一，轮播时应该使每张图片连在一起，我这里用的是ul、li的组合。第二，如果要在包含ul的父元素上添加其他标签时，最好用float，然后清除浮动。第三，图片的自适应，直接设置其max-width为100%即可。
