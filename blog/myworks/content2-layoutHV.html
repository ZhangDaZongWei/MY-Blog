    • 水平与垂直居中布局总结

    张宗伟 平常 更新于 2018-03-07 我的作品 | #html css js

    对于网页布局时用到的水平居中和垂直居中，早已是一个必然的需求，然而这里却有很多技巧或者说坑。在此之前我也使用过很多方法，苦于没有系统的总结，所以用起来心里没谱或者想不起来。就想写一篇文章总结一下。

    实现水平居中

    行内元素/行内块元素/单纯文字

    html代码模板

    <div class='first'><span>我是第一个小片段</span></div>
<div class='second'>我是第二个小片段</div>
<div class='third'>
    <div>我是第三个小片段</div>
</div>

    为相应的标签设置text-align: center规则即可

    css代码模板

        .frist {
        width: 300px;
        height: 100px;
        text-align: center;
        }
    .second {
        width: 300px;
        height: 100px;
        text-align: center;
        }
    .third {
        width: 300px;
        height: 100px;
        text-align: center;
        }
    .third div {
        display: inline-block;
        }

    块级元素

    html代码模板

    <div class='first'>我是第一个块级元素</div>
<div class='second'>
    <div>我是第二个块级元素</div>
</div>
<div class='third'>
    <div>我是第三个块级元素</div>
</div>
  • 为其本身设置margin: 0 auto规则,这个规则需要设置宽度。
  • 为其本身设置 position: absolute; left: -50% margin-left: -(width/2) 即先设置本身为绝对定位，并且定位到父元素宽度一半的位置，最后向左移动本身宽度的一半。注意margin不可以用percentage，否则参考父元素的宽度。另外这个规则最好设置宽度，不然要自己算了。
  • 为其本身设置 position: absolute; left: -50% transform: translateX(-50%) 即先设置本身为绝对定位，并且定位到父元素宽度一半的位置，最后向左移动本身宽度的一半。这个规则和上面师出同门，但是进阶技能不一样。建议使用transform，因为不用管宽度。

    • css代码模板

       .frist {
        width: 300px;
        heigth: 100px;
        margin: 0 auto;
    }
    .second {
        position: relative;
        width: 300px;
        heigth: 100px;
    }
    .second div {
        position: absolute;
        width: 100px;
        left: 50%;
        margin-lsft: -50px;
    }
    .third {
        position: relative;
        width: 300px;
        heigth: 100px;
    }
    .third div {
        position: absolute;
        width: 100px;
        left: 50%;
        transform: translateX(-50%);
    }

    实现垂直居中

    行内元素/行内块元素/单纯文字

    为其父元素设置line-height: (height)规则即可，这里就不再赘述了。注意多行的文字就不行了。

    块内元素

    为其本身设置 position: absolate; top: 50% margin-top: -(height/2) 即先设置本身为绝对定位，并且定位到父元素高度一半的位置，最后向上移动本身高度的一半。同样这个规则最好设置高度；正如你所想的一样，你也可以将margin-top改为 transform: translateY(-50%) 这样就免去你计算高度值。

    使用特定的布局

    我认为这种方式才是强大的实现水平垂直居中的方式，当然一定要注意浏览器的支持程度问题。

    flex布局

    html代码模板

    <div class='frist'>
    <div>我是flex布局</div>
</div>
<div class='second'>
    <div>我是第一个flex布局</div>
    <div>我是第二个flex布局</div>
</div>
    这种布局我之前写过一个网页端手机版 携程首页， 关于flex布局的使用请移步 阮一峰， 切入正题
  • 实现水平居中：justify-content: center即可。
  • 实现垂直居中：align-items: center即可。
  • 实现多行垂直居中：align-content: center即可。

    • css代码模板

     .frist {
        display: -webkit-flex;
        justify-content: center;
        align-items: center;
        width: 300px;
        height: 300px;
        background-color: tomato;
    }
    .frist div {
        width: 100px;
        height: 50px;
        background-color: brown;
    }
    .second {
        display: -webkit-flex;
        flex-wrap: wrap;
        align-content: center;
        width: 300px;
        height: 300px;
        background-color: tomato;
    }
    .second div {
        flex-basis: 100%;
        height: 50px;
        background-color: brown;
    }

    注意，这儿有一个问题，就是在不同行之间会出现一个大的空隙，当然应该是有解决的方法。请参考 阮一峰

    table-cell布局

    html代码模板

    <div class='second'>
    <div>我是table-cell布局</div>
</div>
    这里的思想就是把元素设置为单元格，对于表格天生就好设置水平和垂直居中。关于table-cell的使用请自行搜索。
  • 实现水平居中：text-align: center即可
  • 实现垂直居中，包括多行：vertical-align: middle即可

    • css代码模板

     .frist {
        display: table-cell;
        text-align: center;
        vertical-align: middle;
    }

    图片和文字垂直居中

    给图片设置vertical-align: middle即可。

    再说一下vertical-align: middle这个规则，意为垂直居中，它可以作用于行内，行内块，图片元素，但是我一直用不好它 摊手。
